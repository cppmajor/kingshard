#kingshard简介

kingshard是一个由Go开发高性能MySQL Proxy项目，kingshard在满足基本的读写分离的功能上，致力于简化MySQL分库分表操作；能够让DBA通过kingshard轻松平滑地实现MySQL数据库扩容。

##主要功能：	
	1.读写分离。
	2.跨节点分表。
	3.客户端IP访问控制。
	4.平滑上线DB或下线DB，前端应用无感知。
